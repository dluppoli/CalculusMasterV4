Applicazione demo per laboratori di deploy su Google Cloud Platform.

Applicazione web realizzata con node.js utilizzando express framework e mustache templates. La versione attuale non utilizza database.

Due funzionalità disponibili:

  • Calcolo numeri primi con algoritmo di Eratostene
  • Calcolo cifre pi greco con Unbounded Spigot Algorithm

Seguiranno altre versioni dell'applicazione, con struttura modificata e/o con aggiunta ulteriori funzionalità, al fine di testare altri servizi GCP.
